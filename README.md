# Backend Learning
A backend development practice repository built while learning **Node.js**, **Express.js**, and **MongoDB**. This project focuses on understanding backend architecture, REST APIs, database integration, authentication, and best practices used in real-world applications.

## 🚀 Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JavaScript (ES6+)
* dotenv
* Nodemon

## 📂 Project Structure

```text
src/
├── controllers/
├── db/
│   └── index.js
├── middlewares/
├── models/
├── routes/
├── utils/
├── app.js
├── constants.js
└── index.js
```

## 📚 Learning Objectives

* Understand backend architecture

* Handle errors effectively

* Write clean and maintainable backend code

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/backend-learning.git
```

Navigate to the project directory:

```bash
cd backend-learning
```

Install dependencies:

```bash
npm install
```

Create a `.env` file in the project root:

```env
PORT=8000
MONGODB_URL=mongodb://127.0.0.1:27017
DB_NAME=backend
```

Start the development server:

```bash
npm run dev
```
