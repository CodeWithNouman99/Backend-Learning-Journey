Cloudinary

Cloudinary is a cloud-based service used to store, manage, optimize, and deliver images and videos in web applications.

Interview (1 line):

Cloudinary is a cloud storage service for uploading and serving images and videos.

Multer

Multer is an Express.js middleware used to handle multipart/form-data, which is mainly used for uploading files.

Interview (1 line):

Multer is an Express middleware used to upload files from the client to the server.


Easy way to remember
☁️ Cloudinary → Stores files (images/videos) in the cloud.
📂 Multer → Receives uploaded files from the client and makes them available on the server.


What is fs in Node.js?

fs stands for File System.

It is a built-in Node.js module used to create, read, update, delete, copy, rename, and manage files and folders on your computer.

Interview definition (1 line)

fs (File System) is a built-in Node.js module used to work with files and directories.
